## 操作

npm run build  打包文件  
npm start  启动服务  
npm docs  文件的转化  

## 新增银行配置  
- tsconfig.json 增加ts设置   

## 目录结构

```
├─src                     
├   ├─api                 接口名字
├   ├─Children            子模版
        ├─router          路由
        ├─store           仓库
        ├─view            视图文件（页面管理）
├   ├─Common              公共页面 不要轻易修改
        ├─assets          静态资源
        ├─Plugins         公共模块
        ├─utils           全局工具栏
        ├─view            视图文件（页面管理）
├   ├─component           组建化
├   ├─doxs                转换模版
├   ├─docsTemplate.html   转换模版
├   ├─router              路由配置
├   ├─store               数据管理
├   ├─views               页面管理
```
